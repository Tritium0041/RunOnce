# 一次运行

*[English](./README-en.md)*  

![示例图片](.gif)  

右键, "在此运行"大模型生成的几十百来行代码的"一次性"程序.   

## 安装

[微软商店]()  

程序在安装后会自动注册Windows 11右键菜单项"在此运行代码".  

## 语言支持

支持以下语言:  

- `bat`  
- `powershell`  
- `python`  
- `lua`  
- `nim`  
- `go`  

## UTF-8运行环境  

在中文系统下, 部分语言(如`lua`)默认的运行环境可能造成编码乱码问题.  
在设置中切换`UTF-8`的运行环境以解决这些问题.  

## 模式选择: `确保兼容`和`确保清理`  

此两种模式控制临时文件的生成路径, 对于一般的脚本而言, 使用确保清理模式即可.  
不过, 如果你的脚本有类似`os.path.abspath(__file__)`这样获取脚本真实路径的语句, 你就需要使用确保兼容模式了.  
需要注意的是, 在确保兼容模式下, 临时文件可能因为终端异常关闭等原因无法正确清理残留在目录下.
